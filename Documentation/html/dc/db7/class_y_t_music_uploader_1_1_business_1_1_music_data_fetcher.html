Detailed Description

Gets meta data / tag information from the music file itself and looks up any missing data using a MusicBrainz API implementation (for instance album cover art, in which it will lookup CoverArtArchive: https://coverartarchive.org/

Thanks to: avatar29A https://github.com/avatar29A/MusicBrainz

NOTE: Only 1 request per second allowed via MusicBrainz API, hence plenty of 'Thread.Sleep(1100)' commands.
