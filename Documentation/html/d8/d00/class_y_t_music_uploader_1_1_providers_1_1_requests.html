static UploadCheckResult YTMusicUploader.Providers.Requests.IsSongUploaded ( string  musicFilePath,

MusicDataFetcher  musicDataFetcher = null,
bool  checkCheck = true,
bool  parallel = true 
)
static

HttpWebRequest POST request to send to YouTube to determine if the song about to be uploaded already exists It may request a little attention depending on the results, seen as though we're trying to match a song based on artist, album and track name, which may be slightly different in the uploading music file's meta tags. Currently it uses a Levenshtein distance fuzzy logic implementation to check similarity between strings and is considered a match if it's above 0.75.

Parameters
musicFilePathPath to music file to be uploaded
cookieValueCookie from a previous YouTube Music sign in via this application (stored in the database)
entityIdOutput YouTube Music song entity ID if found
musicDataFetcherYou can pass an existing MusicDataFetcher object, or one will be created if left blank
checkCheckWhether or not to refer to cache for lookup (only useful while scanning)/param>
Returns
True if song is found, false otherwise
