static List<string> YTMusicUploader.Providers.Playlist.Models.M3uPlaylist.GetPlaylistFromCorruptM3u ( string  playlistPath)
static

This handles playlists created in a particular version of VLC and saved as .mp3u. For some odd reason VLC isn't saving them correctly. In fact, you can't even open that playlist in VLC!

None-the-less, the playlist does include the file paths, therefore we can parse them.
