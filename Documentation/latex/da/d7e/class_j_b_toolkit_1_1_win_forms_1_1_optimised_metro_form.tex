\doxysubsection{Detailed Description}
Abstract form to inherit that uses the Metro\+Forms styler, and optimises buffering, drawing, size grips, resizing shadow, allows custom tooltip for each control by putting a string in the control\textquotesingle{}s \textquotesingle{}Tag\textquotesingle{} property and handles client screen D\+PI not set to 100\% more effectively. 

Be sure to call On\+Load(e) / On\+Load(null) in calling Form contructor or \textquotesingle{}Load\textquotesingle{} method.
